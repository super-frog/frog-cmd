### Getting Start !

- 全局安装 **frog**

	```
	npm install -g frog-cmd
	```
	
- 创建项目

	```
	frog create xxx
	
	cd xxx
	
	```
- 安装依赖

   ```
   yarn
   ```
- 构建一下

	```
	frog build
	```
- 运行

	```
	npm start
	```
